QStyleHintReturnMask ¶

Detailed Description ¶

The PySide.QtGui.QStyleHintReturnMask class provides style hints that return a PySide.QtGui.QRegion .
class PySide.QtGui. QStyleHintReturnMask ¶

Constructs a PySide.QtGui.QStyleHintReturnMask . The member variables are initialized to default values.

PySide.QtGui.QStyleHintReturnMask. StyleOptionVersion ¶

This enum is used to hold information about the version of the style option, and is defined for each PySide.QtGui.QStyleHintReturn subclass.

Constant Description
QStyleHintReturnMask.Version 1

The version is used by PySide.QtGui.QStyleHintReturn subclasses to implement extensions without breaking compatibility. If you use qstyleoption_cast() , you normally do not need to check it.

PySide.QtGui.QStyleHintReturnMask. StyleOptionType ¶

This enum is used to hold information about the type of the style option, and is defined for each PySide.QtGui.QStyleHintReturn subclass.

Constant Description
QStyleHintReturnMask.Type The type of style option provided ( SH_Mask for this class).

The type is used internally by PySide.QtGui.QStyleHintReturn , its subclasses, and qstyleoption_cast() to determine the type of style option. In general you do not need to worry about this unless you want to create your own PySide.QtGui.QStyleHintReturn subclass and your own styles.
