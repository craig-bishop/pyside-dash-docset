QWebPage.ErrorPageExtensionOption
*********************************

.. inheritance-diagram:: QWebPage.ErrorPageExtensionOption
    :parts: 2

.. note:: This class was introduced in Qt 4.6
Detailed Description
--------------------



    The :class:`~PySide.QtWebKit.QWebPage.ErrorPageExtensionOption` class describes the option for the error page extension.

    The :class:`~PySide.QtWebKit.QWebPage.ErrorPageExtensionOption` class holds the ``url`` for which an error occurred as well as the associated ``frame`` .

    The error itself is reported by an error ``domain`` , the ``error`` code as well as ``errorString`` .

    .. seealso::  :meth:`QWebPage.extension()<PySide.QtWebKit.QWebPage.extension>`  :class:`QWebPage.ErrorPageExtensionReturn`
