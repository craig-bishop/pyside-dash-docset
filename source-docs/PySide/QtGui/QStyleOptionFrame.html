QStyleOptionFrame¶

Inherited by: QStyleOptionFrameV2, QStyleOptionFrameV3

Detailed Description¶

The PySide.QtGui.QStyleOptionFrame class is used to describe the parameters for drawing a frame.

PySide.QtGui.QStyleOptionFrame is used for drawing several built-in Qt widgets, including PySide.QtGui.QFrame , PySide.QtGui.QGroupBox , PySide.QtGui.QLineEdit , and PySide.QtGui.QMenu . Note that to describe the parameters necessary for drawing a frame in Qt 4.1 or above, you must use the PySide.QtGui.QStyleOptionFrameV2 subclass.

An instance of the PySide.QtGui.QStyleOptionFrame class has type SO_Frame and version 1.

The type is used internally by PySide.QtGui.QStyleOption , its subclasses, and qstyleoption_cast() to determine the type of style option. In general you do not need to worry about this unless you want to create your own PySide.QtGui.QStyleOption subclass and your own styles. The version is used by PySide.QtGui.QStyleOption subclasses to implement extensions without breaking compatibility. If you use qstyleoption_cast() , you normally do not need to check it.

If you create your own PySide.QtGui.QStyle subclass, you should handle both PySide.QtGui.QStyleOptionFrame and PySide.QtGui.QStyleOptionFrameV2 .

For an example demonstrating how style options can be used, see the Styles example.
