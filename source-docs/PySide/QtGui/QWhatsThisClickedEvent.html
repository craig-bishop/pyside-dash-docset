Detailed Description¶

The PySide.QtGui.QWhatsThisClickedEvent class provides an event that can be used to handle hyperlinks in a “What’s This?” text.

class PySide.QtGui.QWhatsThisClickedEvent(href)¶
Parameters:href – unicode

Constructs an event containing a URL specified by href when a link is clicked in a “What’s This?” message.

PySide.QtGui.QWhatsThisClickedEvent.href()¶
