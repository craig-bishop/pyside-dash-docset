QSqlDriverCreatorBase¶

Detailed Description¶

The PySide.QtSql.QSqlDriverCreatorBase class is the base class for SQL driver factories.

Reimplement PySide.QtSql.QSqlDriverCreatorBase.createObject() to return an instance of the specific PySide.QtSql.QSqlDriver subclass that you want to provide.

See QSqlDatabase.registerSqlDriver() for details.
