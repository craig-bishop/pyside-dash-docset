Detailed Description ¶

The PySide.QtCore.QDynamicPropertyChangeEvent class contains event parameters for dynamic property change events.

Dynamic property change events are sent to objects when properties are dynamically added, changed or removed using QObject.setProperty() .

class PySide.QtCore. QDynamicPropertyChangeEvent ( name ) ¶
Parameters: name – PySide.QtCore.QByteArray

Constructs a dynamic property change event object with the property name set to name .

PySide.QtCore.QDynamicPropertyChangeEvent. propertyName ( ) ¶
Return type: PySide.QtCore.QByteArray

Returns the name of the dynamic property that was added, changed or removed.
