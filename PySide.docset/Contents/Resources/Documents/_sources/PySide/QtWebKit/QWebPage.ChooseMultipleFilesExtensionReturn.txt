Detailed Description
--------------------



    The :class:`~PySide.QtWebKit.QWebPage.ChooseMultipleFilesExtensionReturn` describes the return value for the multiple files selection extension.

    The :class:`~PySide.QtWebKit.QWebPage.ChooseMultipleFilesExtensionReturn` class holds the filenames selected by the user when the extension is invoked.
